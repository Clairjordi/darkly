# darkly

Darkly est un projet pédagogique visant à introduire les bases de la cybersécurité dans le contexte du Web.  
Il propose un site vulnérable hébergé sur une machine virtuelle qui contient plusieurs failles à exploiter.


## 1. Objectifs

- Découvrir et comprendre les vulnérabilités web courantes (injections, failles d’authentification, etc.).  
- Se familiariser avec l’**OWASP** et ses risques principaux.
- Comprendre les mécanismes des frameworks web qui peuvent protéger contre certaines vulnérabilités. 
- Produire un travail explicatif : non seulement exploiter les failles, mais documenter clairement le processus d’exploitation et de remédiation.


## 2. Prérequis & lancement

- Une **machine virtuelle**
- Un navigateur web pointant vers l’IP donnée (par exemple `10.13.248.4`) sur le port 80.
